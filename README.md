# MRS - Material Requirement System

A full-stack MERN application that calculates material requirements based on product Bill of Materials (BOM) and order quantity. It helps companies estimate raw material usage, stock availability, and shortages.

---

## 🚀 Features

- Create and manage Materials
- Create Products with BOM (Bill of Materials)
- Manage Companies
- Create Orders for products
- Automatic Material Requirement Summary (MRS)
- Stock vs Requirement calculation
- Real-time population of relational data using MongoDB
